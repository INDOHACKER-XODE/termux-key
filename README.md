## [+] Installation :

* ```apt update```
* ```apt install git -y```
* ```git clone https://github.com/INDOHACKER-XODE/termux-key```
* ```cd termux-key```
* ```bash key.sh```

### Or ; Use Single Command
```
apt update && apt install git -y && git clone https://github.com/INDOHACKER-XODE/termux-key && cd termux-key && bash key.sh
```
